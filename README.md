# Doctor Management System

A desktop-based Doctor Management System developed using Java and Object-Oriented Programming principles. The system helps manage doctors, administrators, and healthcare records through a graphical user interface (GUI).

## Features

- Doctor Management
- Admin Authentication
- Doctor Registration
- Doctor Information Management
- GUI-Based Interface
- Object-Oriented Design

## Technologies Used

- Java
- Java Swing
- OOP Concepts
- File Handling

## Project Structure

- Admin Login Module
- Doctor Management Module
- Dashboard Interface
- Data Storage System

## Learning Outcomes

This project was developed to strengthen understanding of:

- Object-Oriented Programming (OOP)
- Java GUI Development
- Event Handling
- File Management
- Software Design Principles
